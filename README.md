## Dockerの立ち上げ方

1. Dockerを使える状態にする
2. ターミナルを開いて`cd`で`game_collect/docker`まで移動する
3. 下記のコマンドを実行

```sh
docker-compose up -d
```

## Dockerの停止と破棄

```
docker-compose down
```
